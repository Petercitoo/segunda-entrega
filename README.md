# segunda-entrega

# Requisitos
  pip
  python 3.x
  jupyter

# Instalacion
  1. Clona el repo
  2. Crea un entorno virtual
     python -m venv .venv
  
  3. Activalo
     .venv\Scripts\activate
     #windows
     source .venv/bin/activate
     #linux
     (Basta con hacer copia y pega ctrl + V windows, ctrl + shift + V linux)
  4. Instala las dependencias (python venv viene con python y pip):
     pip install jupyter

# Como ejecutar
  1. Lanza jupyter dentro del venv
     jupyter notebook / jupyter lab (Ambos sirven)
  2. Navega hasta la carpeta notebook/ y ejecuta el Inciso de tu eleccion.
